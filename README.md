# Wessel Developer Platform (WDP)

The Wessel Developer Platform (WDP) is a personal software engineering ecosystem designed to accelerate the development of high-quality, production-ready applications.

WDP provides:

- Reusable project templates
- Shared libraries
- Engineering standards
- AI-assisted development workflows
- Documentation templates
- GitHub automation
- Development tooling
- Future CLI automation

The goal is simple:

> Build once.
> Improve forever.
